# CLAUDE.md — Claude가 이 저장소에서 지킬 규칙

이 저장소는 **PUBLIC**이며 대외 공개용 정적 콘텐츠 전용입니다.
보안 규칙 전문은 [SECURITY.md](./SECURITY.md)를 따른다.

## 반드시 지킬 것

1. **push 전 민감정보 스캔** — 커밋/푸시 전 변경 파일에 개인정보·키·토큰·비밀번호·실데이터가 없는지 항상 확인한다. 하나라도 의심되면 push하지 말고 사용자에게 먼저 알린다.
2. **`main` 직접 push 금지** — 반드시 브랜치를 만들고 PR을 통해 병합한다. (`git switch -c ...` → `gh pr create`)
3. **데이터·자격증명 파일 커밋 금지** — `.env`, `*.csv/.xlsx/.sql/.db`, `credentials/secret/token/key` 류 파일은 절대 `git add` 하지 않는다. (`.gitignore`로도 차단되어 있음)
4. **고객/사내 데이터 반입 금지** — 잡코리아·알바몬·나인하이어·잡플래닛·게임잡 등 사내/고객 데이터를 이 저장소로 가져오지 않는다. 관련 작업은 별도 Private 저장소에서 한다.
5. **공개 적합성 자문** — 새 콘텐츠를 올리기 전 "이게 전 세계에 공개돼도 괜찮은가?"를 스스로 확인하고, 애매하면 사용자에게 묻는다.
